# Proyecto-integrador-mezcla-de-unicel-
Nombre del proyecyo:
<h3>Diseño y desarrollo de sistema semi automático de mezcla para la elaboración de bloques ecológicos de cemento y unicel.</h3>

El proyecto se enfoca en hacer una mezcla basada principalmente en unicel para la elaboración de placas o blocks con unicel, haciendo que estas sean ligeras y termoasilantes. Su uso varía desde aplicaciones en fachadas para disminución de temperaturas en inmuebles a una opción de mezcla para uso decorativo, como pisos para jardineras, etc.


El proyecto se divide en 3 sistemas clave:

-Sistema de triturado:
Se encarga de triturar el unicel y alamacenarlo en una tolva para su posterior uso.

-Sistema de mesurado:
Se encarga de cuantificar y dosificar los materiales necesarios para hacer la mezcla.

-Sistema de Mezclado:
Se encarga de mezclar los materiales para obtener la mezcla final.
